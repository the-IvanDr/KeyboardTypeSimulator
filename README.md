## Getting Started (Some instructions)

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Then you will get a link (smth type of https://localhost:3000).

To creat a build:

```bash
npm run build
# or
yarn build
```

And with 'start' script you can run the server with the created build.
